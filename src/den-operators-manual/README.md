# Den Operator's Manual
*A Cozy‑Professional Guide to the Kitsune Den Ecosystem*

Welcome to the Den.
This manual provides a structured, readable, and gently lore‑infused guide to operating, maintaining, and extending the Kitsune Den server ecosystem.

The Den currently manages:
- **The Unified Dashboard** — a Next.js interface overseeing Minecraft (Fabric & NeoForge) and Hytale with live consoles, mod packs, memory management, backups, and player controls

It is designed for:
- Ada (Operator of the Den)
- Sage, Claude, and active collaborators
- Anyone maintaining or extending the Den's dashboards and services

The tone is warm but clear, cozy but professional — a lantern‑lit workshop with proper documentation.
